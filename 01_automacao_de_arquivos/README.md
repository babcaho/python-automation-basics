# Automação de Organização de Arquivos

## O que este projeto faz
Este projeto automatiza a organização de arquivos em pastas, separando-os
por tipo (PDF, imagens, textos, etc.) e gerando um relatório automático com
a quantidade de arquivos organizados.

## Conceitos demonstrados
- Lógica de programação
- Automação de tarefas com Python
- Manipulação de arquivos e pastas
- Geração de relatórios
- Organização de código

## Por que este projeto é relevante
Automação de processos é amplamente utilizada em ambientes corporativos
para aumentar produtividade e reduzir erros manuais. Este projeto demonstra
capacidade de identificar um problema repetitivo e resolvê-lo com código.
