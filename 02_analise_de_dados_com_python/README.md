# Análise de Dados com Python

## O que este projeto faz
Realiza a leitura de um arquivo CSV contendo valores de vendas e calcula
informações importantes como total, média, maior e menor valor.

## Conceitos demonstrados
- Leitura de arquivos CSV
- Análise de dados básica
- Estruturas de repetição
- Operações matemáticas
- Geração de informações a partir de dados

## Por que este projeto é relevante
Transformar dados brutos em informações úteis é uma habilidade essencial
em diversas áreas, como negócios, finanças e tecnologia.
