# Sistema com Banco de Dados SQLite

## O que este projeto faz
Cria um banco de dados SQLite, insere registros e realiza consultas simples
utilizando Python integrado com SQL.

## Conceitos demonstrados
- SQL básico (CREATE, INSERT, SELECT)
- Integração de banco de dados com Python
- Persistência de dados
- Lógica de sistemas CRUD

## Por que este projeto é relevante
Este projeto demonstra a conexão entre código e banco de dados, habilidade
fundamental para desenvolvimento de sistemas reais.
