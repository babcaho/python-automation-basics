# Classificador Simples de Texto (IA Básica)

## O que este projeto faz
Recebe frases digitadas pelo usuário e realiza uma classificação simples
como positiva, negativa ou neutra.

## Conceitos demonstrados
- Fundamentos de Inteligência Artificial
- Classificação de dados
- Lógica condicional
- Simulação de comportamento inteligente

## Por que este projeto é relevante
O projeto demonstra entendimento conceitual de como sistemas de IA funcionam,
indo além do uso de ferramentas prontas.
